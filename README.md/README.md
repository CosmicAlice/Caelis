# PlataStore

PlataStore es una plataforma e-commerce para la venta online de joyería de plata, desarrollada bajo una arquitectura de microservicios con Spring Boot.

## Tecnologías principales

- Java 21
- Spring Boot
- Maven
- MySQL
- Spring Data JPA
- Spring Validation
- OpenFeign
- Eureka
- API Gateway
- Docker
- Git y GitHub

## Microservicios

- ms-auth-usuario
- ms-cliente
- ms-direccion
- ms-categoria
- ms-producto
- ms-inventario
- ms-carrito
- ms-pedido
- ms-pago
- ms-envio
- ms-resena
